Advanced Location Sensor

Product Features
Single board computer running Debian 7.0 with 128 MiB DDR2 SDRAM, 1 GiB Flash, integrated controllers and 6 USB 2.0 High Speed 480Mbits/s ports.
Product Description
The sensor was designed to be the most versatile Inertial Measurement Unit (IMU) and Attitude Heading Reference System (AHRS) platform available. Its host of on-board sensors, algorithms, configurable auxiliary port and real-time communication via USB, Bluetooth or UART make it both a powerful sensor and controller. The on-board SD card* , battery charger (via USB), real-time clock/calendar and motion trigger wake up also make the sensor an ideal standalone data logger. The sensor GUI can be used to configure settings, view real-time sensor data, perform calibration and export data to user software; e.g. Microsoft Excel. The sensor MATLAB library provides all the tools required to import, organise and plot data in MATLAB. User software can be developed using the sensors API.
