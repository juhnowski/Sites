Golf Extension

Description
The general idea is: to use special golf ball gun to create 3D object ib the sky. This feature will be interesting as is like a perfomance, and like tools for new golf ruls extensions.
Golf ball gun is controlled by PLC wich is used video stream and environment sensor.
User may choose objects from presets.
Another user case is: web cam shoot flying ball and create 3D objects as a border. It will be possible to use multi-color balls with one type markers - control balls - for check environments changing.
We are sure that this installation diversifies classical golf and will allow to create new interesting game clones.

Project status

We have demo software and hardware sensors, but we haven't gun in metall, it's needed develop.
If you are interested in our development and you have unlimited access to golf field and equipment for installation testing, please contact with us.
On completion of development and creation of a test sample we need one and a half years.
We will be glad to cooperation with you.

Wireless power installation

Description
There is a need to facilitate of aircraft. We suggest to remove power installation/battery from aircraft.
Having a steady source of energy the aircraft is in the air can so long as it is needed.
The principle of werk of the remote laser power device is shown in the figures.
Installed on the platform hinge with the laser. The laser can be twisted in space. On power device the camera is installed, which accompanies the aircraft. The controller monitors the behavior of the aircraft space and transmits signals to the motion controller which accompanies the object.
In theory, laser system hasn't restrictions on capacity.

Project status

It was tested laser power of 400Wt. In theory, the transmission distance is limited by the energy dissipation in the atmosphere, which can be overcome by increasing the laser power.
Using a single installation makes the restriction on use. Energy is transferred only within visible area. If the laser beam is overlapped by another object or aircraft flies into an invisible zone, the power supply is stopped.
Research is needed to use several power base.
Surveys need to be addressed synchronization and protection from excess capacity.
Also, studies are needed to optimize the conversion mechanism of the laser beam into an electric current. In the current implementation, we use sensors based on conversation of thermal energy into electrical energy.
To design and build a prototype is necessary for one to two years.
We will be glad to cooperation with you.
