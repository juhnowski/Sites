USB2.0 4-Port Hub

Product Features
Supports up to four USB 2.0 and/or 1.1 devices.
Product Description
This ultra small 4-port USB 2.0 allows you to connect up to four USB devices to a single USB port on a PC/Mac/laptop. It meets USB 2.0 Specifications for hi-speed data transmission rates up to 480Mbps. This USB 2.0 device is backwards compatible so that it will function with both USB 2.0 devices as well as USB 1.1 devices. This is a bus powered USB Hub, that describes itself as being self powered in order to let you use USB devices that consume more than the 100mA bus-powered limit. The 3403 does not require an external power supply.
