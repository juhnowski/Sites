SBC3

Product Features
Single board computer running Debian 7.0 with 128 MiB DDR2 SDRAM, 1 GiB Flash, integrated controllers and 6 USB 2.0 High Speed 480Mbits/s ports.
Product Description
The SBC3 is a Single Board Computer with an integrated InterfaceKit 8/8/8. At its most basic, it can be thought of as a Phidget that you connect using a network cable instead of USB. The SBC3 also provides six high-speed ports that allow you to use normal USB Phidgets over its network connection. This can extend the effective range of a SBC from USBâ€™s maximum of 15 feet, to anywhere that your network reaches. The SBC3 exposes an easy to use interface for setting up and running custom applications on-board. This allows the SBC3 to operate autonomously, without the need for a graphical interface or a remote connection at all times. For more advanced users, the SBC is an embedded computer that runs Debian GNU/Linux. We provide full shell access via a built-in SSH server, access to the full Debian package repository, and all of the standard command line tools expected on a modern Linux system. An integrated InterfaceKit 8/8/8 allows you to connect devices to any of 8 analog inputs, 8 digital inputs and 8 digital outputs. It provides a generic, convenient way to interface your PC and SBC with a wide variety of devices and it operates exactly the same way as an external InterfaceKit.
